# 跨境电商图片提示词库

一个单文件静态页面，用于快速生成美国、英国市场常用电商图片提示词。

## 使用

- 本地直接打开 `index.html`
- 或部署到 GitHub Pages 后在线访问
- 平台选择 `TikTok Shop` 时显示 TikTok Shop 专用模板
- 平台选择其他选项时显示 Amazon / 通用电商模板

## 内容

- TikTok Shop 高点击封面图
- TikTok Shop 真人使用种草图
- TikTok Shop 痛点瞬间图
- TikTok Shop 直播间挂车图
- 白底主图
- 生活场景图
- 核心卖点信息图
- 问题解决型图片
- 同类普通款对比图
- 尺寸规格图
- 材质细节特写
- 7 张图整套生成模板

## 注意

- 不要在页面中保存账号、密钥、供应链信息或未公开产品数据。
- GitHub Pages 默认公开，请只放通用提示词工具。
